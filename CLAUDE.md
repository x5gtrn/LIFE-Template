# CLAUDE.md

## Project: LIFE

Personal Operating System with TIL synchronization.

## Build and Scripts

- `./scripts/sync-til.sh`: Sync TIL from remote.
- `./scripts/build-til-index.md`: Rebuild TIL index.
- `./scripts/build-til-digests.sh`: Build daily/weekly digests.
- `Makefile`: Provides `sync-til` command.

## Tech Stack

- Bash scripts
- GitHub Actions
- Markdown for documentation

## Guidelines

- Keep private information in `LIFE` repo and public notes in `TIL` repo.
- Weekly digests start on Monday 04:00 JST.
- Use English for comments and documentation.
