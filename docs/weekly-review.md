# Weekly Review (Checklist)

## 1) Clear inbox (10 min)
- [ ] Close done Issues
- [ ] Move remaining tasks to correct status
- [ ] Add labels (area / priority)
- [ ] Convert vague notes into next actions

## 2) Look back (10–15 min)
- [ ] Read last week's journal entries
- [ ] Read `logs/learning/learning-log.md`
- [ ] Check this week's TIL weekly digest in `docs/til-weekly/`
- [ ] Identify 3 wins, 3 lessons

## 3) Plan next week (10–15 min)
- [ ] Choose 1–3 priorities (must be small & finishable)
- [ ] Pick 3–5 supporting tasks
- [ ] Decide "Not doing" list (important)

## 4) Maintenance (optional)
- [ ] Archive old notes
- [ ] Update routines or budget if needed
